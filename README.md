# GanaControl - Backend

Backend de sistema web para la gestión de fincas ganaderas, desarrollado con Node.js, Express y MySQL. Implementa una API REST que permite administrar procesos clave del negocio ganadero como inventario, salud, reproducción y ventas.

## Tecnologías
- Node.js
- Express
- MySQL / MariaDB
- Sequelize

## Funcionalidades
- Gestión de ganado
- Módulo de salud animal
- Módulo de reproducción
- Gestión de inventario
- Registro y control de ventas
- Sistema de autenticación
- Control de roles y permisos
- Validación de lógica de negocio

## Instalación
```bash
npm install
npm run dev
```

## Estructura
- config → configuración del sistema
- models → modelos de base de datos
- src → rutas, controladores y lógica del backend
